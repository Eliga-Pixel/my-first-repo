# Linux Commands

## ls
List files in a directory

### Examples
ls
ls -la
ls /home

### Notes
- `-l` shows detailed list
- `-a` shows hidden files

---

## cd
Change directory

### Examples
cd /home
cd ..
cd ~

### Notes
- `..` goes up one level
- `~` goes to home directory

---

## pwd
Print working directory
mkdir
rm
cp
mv
cat
touch

### Example
pwd
